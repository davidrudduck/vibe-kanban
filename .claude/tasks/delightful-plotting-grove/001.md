# Task: Add i18n Translation Keys for Injected Indicator

## Description
Add internationalization support for the injected message indicator by adding `injectedLabel` and `injectedTooltip` translation keys to all 4 locale files (en, ja, ko, es) in the `conversation` namespace.

## Acceptance Criteria
- [ ] English translations added to `frontend/src/i18n/locales/en/common.json`
- [ ] Japanese translations added to `frontend/src/i18n/locales/ja/common.json`
- [ ] Korean translations added to `frontend/src/i18n/locales/ko/common.json`
- [ ] Spanish translations added to `frontend/src/i18n/locales/es/common.json`
- [ ] All JSON files have valid syntax (no trailing commas, proper quoting)
- [ ] Keys are in the `conversation` namespace

## Technical Details
- Implementation approach: Add 2 keys to each locale file
- Namespace: `common.json` -> `conversation` section
- Keys:
  - `injectedLabel`: "(injected)" / localized equivalent
  - `injectedTooltip`: "This message was injected into the running process" / localized equivalent

### Files to Modify
| File | Changes |

| `frontend/src/i18n/locales/en/common.json` | Add 2 keys to conversation namespace |
